解决Django与echarts结合出现Uncaught SyntaxError: Unexpected token &和&#39;的问题

2020-03-30 echarts ， 解决问题 ， Django

出现的问题

在Django与echarts结合中，出现了
data:[&#39];和Uncaught SyntaxError: Unexpected token &的问题

出错的代码

    xAxis: {
        type: 'category',
        boundaryGap: false,
        data: {{ name_list}}
    },

原因是前端页面echarts接收数据时，没有添加|safe

解决的方法

在echarts页面data接收来自views.py的数据时，加上|safe,具体效果如下：

    xAxis: {
        type: 'category',
        boundaryGap: false,
        data: {{ name_list|safe }}
    },

重启Django应用即可正确显示echarts图表
